설치 전 필수 확인 사항
  • iPhone에 SideStore 앱이 미리 설치되어 있고, 첫 Refresh까지 완료되어 있어야 합니다.
  • 이 페이지는 무료 Apple ID 개발 계정을 기준으로 설계되었습니다. 설치된 앱은 7일마다 SideStore 앱을 통해 만료를 갱신(Refresh)해야 작동을 유지합니다.
  • 정적 GitHub Pages 방식에서는 로그인 화면이 테스트용 잠금입니다. 링크를 아는 사용자는 파일 URL에 직접 접근할 수 있습니다.
  • 이 사이트의 admin / admin8579 로그인은 웹사이트용입니다. SideStore에는 Apple ID로 로그인해야 합니다.
  • iOS 보안 정책상 웹사이트에서 곧바로 앱 설치가 완료되지 않으며, 링크를 누르면 SideStore 앱으로 전환되어 최종 설치를 승인해야 합니다.
  • 앱 설치, 앱 업데이트, Refresh 전에는 iPhone에서 LocalDevVPN이 켜져 있어야 합니다.

SideStore 정확한 사용 순서

1. 처음 1번만 Mac에서 SideStore 설치

  1. iPhone에 비밀번호가 설정되어 있고 iOS/iPadOS 15 이상인지 확인합니다.
  2. iPhone에서 App Store의 LocalDevVPN을 설치하고 한 번 열어 VPN 추가를 허용합니다.
  3. Mac에 iloader를 설치합니다. 현재 공식 SideStore 설치 흐름은 iloader 기준입니다.
  4. iPhone을 USB 케이블로 Mac에 연결하고, iPhone에 이 컴퓨터를 신뢰가 뜨면 신뢰를 누릅니다.
  5. Mac에서 iloader를 열고 Apple ID로 로그인합니다. 2단계 인증이 뜨면 iPhone에 표시된 6자리 코드를 입력합니다.
  6. iloader에서 연결된 iPhone을 선택한 뒤 Install SideStore (Stable)을 실행합니다.

2. iPhone에서 SideStore 활성화

  1. 설정 > 일반 > VPN 및 기기 관리에서 Apple ID 개발자 앱을 선택하고 신뢰를 누릅니다.
  2. iOS 16 이상이면 설정 > 개인정보 보호 및 보안 > 개발자 모드를 켠 뒤 iPhone을 재시동합니다.
  3. LocalDevVPN을 열고 Connect를 눌러 VPN을 켭니다.
  4. SideStore를 열고 iloader에서 사용한 Apple ID로 로그인합니다.
  5. My Apps 탭에서 SideStore 오른쪽의 7 DAYS를 눌러 첫 Refresh를 완료합니다.
  6. 인증서를 새로 만들거나 취소하라는 안내가 뜨면 Yes 또는 Refresh Now를 선택합니다.

3. 이 앱 스토어 소스 추가

  1. iPhone Safari에서 https://jjs111190.github.io/my-ios-store/를 엽니다.
  2. 웹사이트 로그인에 admin / admin8579를 입력합니다.
  3. 상단의 SideStore에 소스 추가하기 버튼을 누릅니다.
  4. 버튼이 동작하지 않으면 SideStore의 Sources 탭에서 +를 누르고 아래 주소를 직접 붙여넣습니다.
  5. https://jjs111190.github.io/my-ios-store/apps.json

4. 앱 설치

  1. iPhone에서 LocalDevVPN이 켜져 있는지 확인합니다.
  2. 웹사이트 앱 카드의 SideStore 설치 버튼을 누르거나, SideStore의 Sources 탭에서 앱을 선택합니다.
  3. SideStore로 화면이 넘어가면 Install을 눌러 설치를 승인합니다.
  4. 홈 화면에 앱이 생기면 실행합니다. 처음 실행이 막히면 VPN 및 기기 관리에서 개발자 앱 신뢰 상태를 다시 확인합니다.

5. 7일 만료 관리

  1. 무료 Apple ID로 설치한 앱은 보통 7일마다 Refresh가 필요합니다.
  2. 만료 전에 LocalDevVPN을 켜고 SideStore > My Apps > Refresh All을 누릅니다.
  3. SideStore 자체가 만료되어 열리지 않으면 Mac에서 iloader로 SideStore를 다시 설치합니다.
  4. iPhone 업데이트, 초기화, 기기 교체 후에는 페어링이 깨질 수 있으므로 iloader에서 pairing file을 다시 등록합니다.

6. 오류가 날 때 확인 순서

  1. No Wi-Fi or VPN: Wi-Fi와 LocalDevVPN이 모두 켜져 있는지 확인하고 SideStore와 LocalDevVPN을 재시작합니다.
  2. 소스 추가 실패: Safari에서 열었는지, SideStore가 설치되어 있는지, 소스 주소가 https://로 시작하는지 확인합니다.
  3. 설치 실패: SideStore 로그인, LocalDevVPN 연결, 남은 앱 슬롯, 7일 만료 여부를 확인합니다.
  4. 앱 실행 불가: Refresh가 만료된 상태일 수 있습니다. SideStore가 열리면 Refresh All, 열리지 않으면 iloader로 SideStore를 재설치합니다.
핵심 순서: LocalDevVPN 켜기 → SideStore 열기 → 소스 추가 → 앱 설치 → 7일마다 Refresh All입니다.
