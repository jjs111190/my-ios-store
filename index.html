설치 전 필수 확인 사항
  • iPhone에 SideStore 앱이 미리 설치되어 있고 활성화되어 있어야 합니다.
  • 이 페이지는 무료 Apple ID 개발 계정을 기준으로 설계되었습니다. 설치된 앱은 7일마다 SideStore 앱을 통해 만료를 갱신(Refresh)해야 작동을 유지합니다.
  • 정적 GitHub Pages 방식에서는 로그인 화면이 테스트용 잠금입니다. 링크를 아는 사용자는 파일 URL에 직접 접근할 수 있습니다.
  • iOS 보안 정책상 웹사이트에서 곧바로 앱 설치가 완료되지 않으며, 링크를 누르면 SideStore 앱으로 전환되어 최종 설치를 승인해야 합니다.
  • 이 저장소는 개인 소유 테스트용 서버입니다. 외부 공유 시 로컬 IP나 터널 주소가 노출될 수 있으므로 허용되지 않은 사용자에게 공개하지 마십시오.

SideStore 완전 사용 가이드

1. iPhone 준비

  1. SideStore 공식 설치 안내 기준으로 LocalDevVPN을 준비합니다. 구버전 안내를 따르는 경우에만 WireGuard/StosVPN을 사용합니다.
  2. Mac에서 SideStore 공식 사이트의 최신 SideStore IPA를 받은 뒤 AltServer 또는 SideServer로 iPhone에 설치합니다.
  3. iOS 16 이상이면 설정 > 개인정보 보호 및 보안 > 개발자 모드를 켜고 재부팅합니다.
  4. 설정 > 일반 > VPN 및 기기 관리에서 Apple ID 개발자 앱을 신뢰합니다.

2. 페어링 파일과 VPN

  1. SideStore 설치 과정에서 만든 .mobiledevicepairing 파일을 AirDrop 또는 파일 앱으로 iPhone에 보냅니다.
  2. SideStore 첫 실행 시 해당 페어링 파일을 선택합니다.
  3. 앱 설치와 Refresh 전에는 LocalDevVPN 또는 설정한 SideStore용 VPN을 켭니다.
  4. SideStore의 Settings 탭에서 Apple ID로 로그인한 뒤 My Apps에서 Refresh All이 되는지 확인합니다.

3. 이 스토어에서 설치

  1. GitHub Pages에 올라간 이 페이지를 iPhone Safari에서 열고 admin 계정으로 로그인합니다.
  2. 앱을 추가하거나 IPA를 교체한 뒤에는 npm run update-url https://사용자명.github.io/저장소명과 npm run generate를 실행하고 GitHub에 push합니다.
  3. 상단의 SideStore 소스 주소를 복사하거나 소스 추가 버튼을 눌러 SideStore Sources에 등록합니다.
  4. 앱 카드의 SideStore 설치 버튼을 누르거나 SideStore의 Sources 탭에서 앱을 선택해 설치합니다.

4. 7일 만료 관리

  • 무료 Apple ID로 설치한 앱은 보통 7일마다 Refresh가 필요합니다.
  • 만료 전에 LocalDevVPN 또는 SideStore용 VPN을 켜고 SideStore에서 Refresh All을 누릅니다.
  • SideStore 자체가 만료되면 Mac에 연결해 AltServer 또는 SideServer로 SideStore를 다시 설치해야 합니다.
설치가 실패하면 먼저 LocalDevVPN/VPN 연결, SideStore 로그인, 페어링 파일, 소스 URL의 HTTPS 주소와 토큰 포함 여부를 확인하세요.
